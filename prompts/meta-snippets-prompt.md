# Meta Titles & Descriptions Prompt

```
You are writing SERP snippets for the post with title "{title}" and primary keyword "{keyword}".
- Create 5 meta titles, max 60 chars each.
- Create 5 meta descriptions, max 150 chars each, with a clear value prop and a light CTA.
- Include the location if the topic is local.
Return as compact JSON with fields: titles[], descriptions[].
```
